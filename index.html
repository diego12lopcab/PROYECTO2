DEFINICION

ASP (Active Server Pages) es una tecnología creada por Microsoft para la creación de sitios web dinámicos. Con ASP el desarrollo web utiliza Visual Basic Script o JScript (una versión similar a JavaScript de la propia Microsoft) para dotar a los sitios web de funciones dinámicas.

Un lenguaje del lado del servidor es aquel que se ejecuta en el servidor web, justo antes de que se envíe la página a través de Internet al cliente.

Las páginas que se ejecutan en el servidor pueden realizar accesos a bases de datos, conexiones en red, y otras tareas para crear la página final que verá el cliente.

El cliente solamente recibe una página con el código HTML resultante de la ejecución de la página ASP. Como la página resultante contiene únicamente código HTML, es compatible con todos los navegadores.

CARACTERISTICAS DE ASP

  • Desarrollo web basado en formularios web.
  • Funcionamiento del lado del servidor (sobre servidores de Microsoft IIS o Apache).
  • Basado en el modelo code-behind (el código está en un archivo separado o en una etiqueta especial), para separar el contenido de la presentación.
  • Controles de usuario para la creación de componentes reutilizables.
  • Acceso a las aplicaciones ASP net alojadas en el servidor web a través del protocolo de navegación de internet HTTP/HTTPS.
  • Es un marco orientado a objetos.
  • Puede utilizarse con sistemas operativos Linux, Windows y macOS.
  • Los errores de compilación son fácilmente depurados.

VENTAJAS Y DESVENTAJAS

Ventajas

  • Permite la creación de aplicaciones web dinámicas de manera relativamente sencilla.
  • Integra bien con otros productos de Microsoft, como SQL Server y componentes COM.
  • Soporta múltiples lenguajes de scripting.

Desventajas

  • El código mezclado con HTML puede dificultar el mantenimiento a medida que la aplicación crece.
  • Comparado con tecnologías más modernas, ASP clásico puede ser menos eficiente y seguro.
  • No es tan escalable como ASP.NET, su sucesor, que introduce un marco de trabajo más robusto y estructurado.
