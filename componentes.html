principales componetes

Motor de Scripts de ASP

  • Es el componente que interpreta y ejecuta los scripts contenidos en las páginas ASP.
  • ASP soporta varios lenguajes de scripting, siendo VBScript y JScript (una variante de JavaScript) los más comunes. El motor de scripts se encarga de procesar estos scripts y generar el contenido dinámico que se envía al cliente.

Proceso de solicitud y respuesta

    Solicitud HTTP:

  • Un usuario solicita una página ASP a través de su navegador.

    • IIS recibe la solicitud:

  • IIS identifica que el archivo solicitado es una página ASP y lo envía al motor de scripts para ser procesado.

    • Ejecución de scripts:

  • El motor de scripts de ASP procesa el código del lado del servidor en la página ASP, interactuando con los objetos ASP y ejecutando cualquier lógica definida (como acceso a bases de datos).

    • Generación de contenido dinámico:

  • El resultado de la ejecución de los scripts es contenido HTML que se envía al navegador del usuario a través de IIS.

    • Respuesta HTTP:

  • envía la respuesta generada al navegador del cliente, que la renderiza como una página web
